Entertaining Mines

Welcome to Entertaining Mines!

Goal of the game

The board consists of a grid of tiles, with a predetermined amount of mines. Lying under any tile could be a mine, and your goal is to find all the mines.

Finding the mines

Every time you click on a tile, as long as the tile doesn't contain a mine, a number will appear in its place.

The number inside a tile indicates how many mines are neighbouring that tile. For example, in this instance, there is one mine neighbouring this tile, so any one of the marked tiles could contain a mine.

Game controls

Entertaining Mines can be controlled with a mouse, a keyboard or a gamepad.

Using a mouse

Hover the mouse over a tile, and then perform one of the following actions:

LEFT CLICK to reveal the number (or mine) under the tile
RIGHT CLICK to flag a tile if you think it contains a mine
MIDDLE CLICK on a already revealed tile to sweep around it. (More on this in the "Sweeping tiles" section)

Using a keyboard

Use the arrow keys to highlight a tile, and then press a key to perform one of the following actions:

ENTER to reveal the number (or mine) under the tile
SPACE to flag a tile if you think it contains a mine
ENTER on a already revealed tile to sweep around it. (More on this in the "Sweeping tiles" section)

Using a gamepad

Use the left stick or the directional buttons to highlight a tile, and then press a button to perform one of the following actions:

to reveal the number (or mine) under the tile
to flag a tile if you think it contains a mine
on a already revealed tile to sweep around it. (More on this in the "Sweeping tiles" section)

Flagging tiles

After flagging a tile, a flag will appear in its place. Flag a tile if you think that a mine is situated on that tile.

INFORMATION
You can't accidentally reveal a tile that you've flagged.

By default, attempting to flag an already flagged tile will turn it into a marked tile. Marked tiles have a ? on them. You can use marked tiles if you are uncertain of where a mine is and you want to come back to it later.

Sweeping tiles

Once a particular tile is satisfied, you are able to sweep it.

INFORMATION
A satisfied tile has exactly the same number of flags neighbouring it as the number on itself.

Sweeping a tile opens all the neighbouring tiles, except for the flagged ones. This can be faster than manually opening all the neighbouring tiles.

DANGER
Be absolutely sure you've put the flags in the right places before you sweep because if any one flag is incorrectly placed, you will uncover a mine and lose the game.
